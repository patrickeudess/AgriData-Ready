9 étapes pour des données prêtes

L'application vous guide pas à pas, de l'import à l'export final

1

Importer les données

Déposez un fichier CSV, Excel ou JSON. L'application affiche un aperçu immédiat : lignes, colonnes, types, valeurs manquantes.

2

Analyser la qualité

Détection automatique des problèmes : valeurs manquantes, doublons, colonnes vides, noms mal écrits, formats incohérents, valeurs anormales.

3

Modifier les données

Éditez directement dans l'application : modifier une cellule, renommer ou supprimer des colonnes, traiter les valeurs manquantes, supprimer les doublons.

4

Documenter les variables

Pour chaque colonne : description, type de donnée, unité, format attendu, valeurs possibles, règle de validation, utilité pour l'IA.

5

Compléter les métadonnées

Titre, auteur, organisation, pays, zone géographique, période de collecte, licence, méthode, contact, citation recommandée.

6

Évaluer la conformité FAIR

Score sur 100 réparti en 4 principes : Findable, Accessible, Interoperable, Reusable. Avec le détail de ce qui manque.

7

Évaluer l'AI Readiness

Score sur 100 évaluant 7 dimensions : complétude, cohérence, documentation, qualité des variables, variable cible, structure, données sensibles.

8

Recommandations

Actions concrètes hiérarchisées par priorité : urgent, important, amélioration. Chaque recommandation est directement actionnable.

9

Exporter le dossier final

Téléchargez un ZIP complet contenant toutes les pièces nécessaires pour un jeu de données documenté et exploitable.

Installation

Lancez l'application en quelques secondes

# Cloner le dépôt
git clone https://github.com/patrickeudess/AgriData-Ready.git
cd AgriData-Ready

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
streamlit run app.py
